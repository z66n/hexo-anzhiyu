记录一下搭建这个博客的文档

## 快速开始

### 安装Hexo

局部安装`hexo`
```bash
npm install hexo
```

把局部安装的执行文件目录加入环境变量
```bash
echo 'PATH="$PATH:./node_modules/.bin"' >> ~/.profile
```

### 安装安知鱼主题

```bash
git clone -b main https://github.com/anzhiyu-c/hexo-theme-anzhiyu.git themes/anzhiyu
```

或者，如果你的博客也在一个git仓库里
```bash
git submodule add https://github.com/anzhiyu-c/hexo-theme-anzhiyu.git themes/anzhiyu
```

把Hexo根目录下的`config.yml`里的`theme`改成`anzhiyu`
```yml
  theme: anzhiyu
```

#### 安装pug和stylus

```bash
npm install hexo-renderer-pug hexo-renderer-stylus --save
```

#### 覆盖配置

```bash
cp -rf ./themes/anzhiyu/_config.yml ./_config.anzhiyu.yml
```

### 写博客

```bash
hexo new "博客文章"
```
然后去`source/_posts/`找到刚刚创建的markdown文件，用你喜欢的文本编辑器写博客

结束之后，构建博客
```bash
npm run build
```

把Hexo根目录里的所有内容复制到nodejs运行的服务器根目录
```bash
cp -rf ./* ~/domains/example.com/public_nodejs/
```
